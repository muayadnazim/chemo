# 🏥 موسوعة الأدوية الكيمياوية - Chemo Drug Encyclopedia

دليل شامل للممرضين والممارسين الصحيين يحتوي على معلومات أكثر من **40 دواء كيمياوي، علاج موجه، علاج مناعي، علاج هرموني، وأدوية داعمة**.

## 📱 المميزات

- ✅ أكثر من 40 دواء مع معلومات كاملة
- 🔍 بحث سريع بالعربية والإنجليزية
- 🏷️ تصفية حسب الفئة (كيمياوي، موجه، مناعي، هرموني، داعم)
- 📲 **يدعم التثبيت على الهاتف** (PWA) ويعمل بدون إنترنت
- 🎨 تصميم متجاوب مع جميع الأجهزة
- 🌐 متوفر باللغة العربية
